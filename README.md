## Features

- Fully responsive layout
- Bootstrap 5 styling
- AOS animations
- Project carousel
- Contact form using Formspree
- Dynamic current year in footer

## Built With

- React
- Bootstrap 5
- Devicon
- Formspree
- AOS (Animate on Scroll)

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

```bash
git clone https://github.com/your-username/personal-portfolio-react.git
cd personal-portfolio-react
npm install
npm start
```
